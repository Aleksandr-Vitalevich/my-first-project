# ___Связываем удаленный репозиторий с локальным___
---

## ___Действия на сайте___
---

1. В строке рядом с профилем нажимаем `+`
2. Выбираем пункт `new repository`
3. Задаем имя. Как правило имя соответствует локальному репозиторию.
4. Выбираем тип доступа
	- `public` 
	- `private`
5. Нажимаем кнопку `create repository`
6. В открывшемся окне выбираем пункт `ssh` (как правило он расположен в верхней строчке выделен синим )
7. Копируем путь

## ___Действия в терминале___
---

1. Связываем локальную папку и репозиторий открываем терминал по адресу папки
2. Проверяем связь локальной папки и удаленного репозитория , команда `git remote -v`
3. Для создания связи `git remote add origin вставляем наш скопированный текст`
4. Примечания к 3 пункту
	- имя origin может быть любым
	- текст вставляем без дополнительных ковычек и прочего. 
5. После можем убедиться в созданной нами связи команда `git remote -v`
6. Для отправки данных используем команду `git push -u origin main`
7. Терминал уточняет мы отправляем данные или нет выбираем
	- yes
	- no
