# ___Создание Fork___
---

## ___Сайт GitHub___
---

1. В GitHub выбираем проект нажимаем `fork`
2. Нажимаем кнопку `create fork`
3. Нажимаем кнопку `code`
4. В открывшемся меню удостоверились , что находимся в поле `ssh` копируем путь

## ___Терминал___
---

1. Выбираем место куда хотим сохранить скопированный проект.
2. Открываем терминал и вводим команду `git clone вставляем скопированный текст`

## ___Командная разработка___

> Если копия проекта устарела необходимо ее обновить для этого используем команду `git pull`
> Если требуется отменить коммит в удаленном репозитории используем команду `git revert хэш коммита`
> Если требуется добавить файл в игнорируемый используем команду `git rm --cached имя файла`
