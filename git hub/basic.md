# ___Инструкция по регистрации на github___
---

1. Заходим на сайт [сайт](https://github.com "github")
2. Нажимаем Sign up (регистрация)
3. Заполняем форму регистрации

# ___Генерация нового SSH-ключа___
---

1. Выполним команду в терминале ssh-keygen
2. Нас пару раз спросят по настройкам
3. Строчка которая нам нужна начинается на Your public ... копируем с `/.ssh/` до конца
4. Вводим команду `cat~` и вставляем скопированный нами текст нажимаем Enter
5. Копируем появившийся текст

# ___Добавление ssh ключа в профиль на github___
---

1. В профиле переходим в настройки.
2. Находим пункт SSH and GPG keys
3. Выбираем New SSH key 
	- в заголовке указываем название для ключа 
	- во второе поле вставляем скопированный ключ
4. Нажимаем кнопку Add SSH key
