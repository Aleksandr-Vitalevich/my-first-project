## ___Команды___
---

1. Нужно включить папку в отслеживаемые GIT
	> Команда `git init` позволяет создать в папке локальный репозиторий
2. Проверка файлов на изменения
	> Команда `git status` позволяет проверить изменения
3. Добавление файла в отслеживаемые
	> Команда `git add имя файла` 
	> Команда `git add -A` сразу несколько файлов
4. Создание коммита
	> Команда `git commit -m "Что изменили"` в ковычках добавляем описание<br>
	> Для первого коммита как правило выбирают `git commit -m "initial commit`
5. Изменение последнего коммита или добавление последних изменений в него
	> Команда `git commit --amend -m "Что изменили"`

## ___Команды___
---

1. Чтобы посмотреть историю изменений по проекту 
	> Команда `git log ` покажет всю историю в развернутом виде<br>
	> Команда `git log --oneline` покажет всю историю но в сжатом виде<br>
	> Команда `git log --oneline --all` покажет всю историю включая и все файлы<br> 
	> Пример Мы находимся в старой или другой ветке после нее были внесены новые изменения данная команда нам покажет их все. Ключ main показывает основную ветку ключ head там где мы находимся
2. Перемещение между коммитами
	> Команда `git checkout хэш` Откатит нас на ту метку которую мы укажем<br>
	> Команда `git checkout main` Вернет нас к последнему коммиту<br>
