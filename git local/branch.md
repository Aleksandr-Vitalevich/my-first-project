# ___Ветки в git___
---

- Для того , чтобы проверить в какой ветке мы находимся используем команду `git branch`
- Для создания новой ветки используем команду `git branch имя ветки`
- Для переключения с одной ветки на другую используем команду `git checkout имя ветки`
- Для отправки ветки в github используем команду `git push -u origin имя ветки`
- Для просмотра изменений в конкретной ветке команда `git log имя ветки`
- Для слияния веток используем команду `git merge имя ветки`
	- > Важно находится в той ветке в которую мы хотим внести изменения

## ___Сокращенные команды___
---

- Для одновременного создания новой ветки и переключения на нее используем команду `git checkout -b имя ветки`
- Для добавления файла в отслеживаемые и его коммита используем команду `git commit -a -m "указываем коммит"`
